## Quick Install

```bash
pnpm add eden-ui
```

```svelte
<script>
  import { Button, Card, Alert } from 'eden-ui';
</script>

<Card color="primary" shadow="lg">
  <Alert color="green">Welcome to Eden UI!</Alert>
  <Button color="primary">Get Started</Button>
</Card>
```
