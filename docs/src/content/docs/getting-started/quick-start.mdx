# Quick Start

Let's build a simple dashboard card using Eden UI components.

## What We're Building

A stats card that displays key metrics with an icon, value, and trend indicator.

## Implementation

<Steps>

1. **Create the Component**

   ```svelte title="src/lib/components/StatsCard.svelte"
   <script lang="ts">
     import { Card, Badge } from 'eden-ui';

     interface Props {
       title: string;
       value: string | number;
       trend?: number;
       icon?: any;
     }

     let { title, value, trend = 0, icon: Icon }: Props = $props();

     const trendColor = $derived(trend >= 0 ? 'green' : 'red');
     const trendSign = $derived(trend >= 0 ? '+' : '');
   </script>

   <Card color="primary" size="md" class="relative overflow-hidden">
     <div class="flex items-start justify-between">
       <div>
         <p class="text-sm text-gray-500 dark:text-gray-400">{title}</p>
         <p class="mt-1 text-3xl font-semibold">{value}</p>

         {#if trend !== 0}
           <Badge color={trendColor} class="mt-2">
             {trendSign}{trend}%
           </Badge>
         {/if}
       </div>

       {#if Icon}
         <div class="p-3 rounded-lg bg-primary-100 dark:bg-primary-900/30">
           <Icon class="w-6 h-6 text-primary-600 dark:text-primary-400" />
         </div>
       {/if}
     </div>
   </Card>
   ```

2. **Use in a Page**

   ```svelte title="src/routes/dashboard/+page.svelte"
   <script>
     import StatsCard from '$lib/components/StatsCard.svelte';
     import { UsersIcon, ShoppingCartIcon, DollarSignIcon } from 'lucide-svelte';
   </script>

   <div class="grid grid-cols-1 md:grid-cols-3 gap-6 p-6">
     <StatsCard
       title="Total Users"
       value="12,543"
       trend={12.5}
       icon={UsersIcon}
     />

     <StatsCard
       title="Orders"
       value="1,234"
       trend={-3.2}
       icon={ShoppingCartIcon}
     />

     <StatsCard
       title="Revenue"
       value="$48,294"
       trend={8.1}
       icon={DollarSignIcon}
     />
   </div>
   ```

3. **Result**

   You now have a responsive dashboard with three stat cards showing:
   - User count with positive trend (green badge)
   - Order count with negative trend (red badge)
   - Revenue with positive trend (green badge)

</Steps>

## Key Concepts

### Props with $props()

Svelte 5 uses `$props()` runes instead of `export let`:

```svelte
// Svelte 5 (Eden UI)
let { color, size }: Props = $props();

// Instead of Svelte 4
export let color;
export let size;
```

### Derived State

Use `$derived()` for computed values:

```svelte
const trendColor = $derived(trend >= 0 ? 'green' : 'red');
```

### Component Composition

Eden UI components are designed to compose together:

```svelte
<Card>
  <Badge>New</Badge>
  <Button>Action</Button>
</Card>
```
