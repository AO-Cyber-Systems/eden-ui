# Theming

Eden UI uses a flexible theming system built on Tailwind CSS and tailwind-variants.

## Customizing Tailwind

### Extending the Color Palette

Add custom colors in your `tailwind.config.js`:

```js title="tailwind.config.js"
export default {
  theme: {
    extend: {
      colors: {
        brand: {
          50: '#f0f9ff',
          100: '#e0f2fe',
          200: '#bae6fd',
          300: '#7dd3fc',
          400: '#38bdf8',
          500: '#0ea5e9',
          600: '#0284c7',
          700: '#0369a1',
          800: '#075985',
          900: '#0c4a6e',
          950: '#082f49',
        }
      }
    }
  }
}
```

### Typography Customization

Override default fonts:

```js title="tailwind.config.js"
export default {
  theme: {
    fontFamily: {
      sans: ['Inter', 'system-ui', 'sans-serif'],
      mono: ['JetBrains Mono', 'monospace'],
    }
  }
}
```

## Component-Level Theming

### Using tailwind-variants

Eden UI components use tailwind-variants for styling. You can extend component themes:

```typescript title="src/lib/theme/button.ts"
import { button } from 'eden-ui/theme';
import { tv } from 'tailwind-variants';

// Extend the base button theme
export const customButton = tv({
  extend: button,
  variants: {
    color: {
      // Add your custom brand color
      brand: 'bg-brand-500 text-white hover:bg-brand-600',
    }
  }
});
```

### CSS Custom Properties

Override CSS variables for global changes:

```css title="src/app.css"
:root {
  /* Override focus ring color */
  --eden-focus-ring: theme('colors.brand.500');

  /* Override default border radius */
  --eden-radius-default: 0.5rem;
  --eden-radius-lg: 0.75rem;
}

.dark {
  --eden-focus-ring: theme('colors.brand.400');
}
```

## Dark Mode

Eden UI supports dark mode via Tailwind's class strategy.

### Manual Toggle

```svelte
<script>
  import { Button } from 'eden-ui';

  let isDark = $state(false);

  function toggle() {
    isDark = !isDark;
    document.documentElement.classList.toggle('dark', isDark);
    localStorage.setItem('theme', isDark ? 'dark' : 'light');
  }
</script>

<Button onclick={toggle}>

</Button>
```

### System Preference

```svelte title="src/routes/+layout.svelte"
<script>
  import { browser } from '$app/environment';

  if (browser) {
    const stored = localStorage.getItem('theme');
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;

    if (stored === 'dark' || (!stored && prefersDark)) {
      document.documentElement.classList.add('dark');
    }
  }
</script>
```

<Aside type="note">
  The `dark` class must be applied to the `<html>` element for Tailwind's dark mode to work correctly.
</Aside>

## Component Size Variants

Most components support consistent size variants:

| Size | Description |
|------|-------------|
| `xs` | Extra small - compact UI |
| `sm` | Small - reduced padding |
| `md` | Medium (default) |
| `lg` | Large - increased padding |
| `xl` | Extra large - prominent UI |

```svelte
<Button size="xs">Tiny</Button>
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
<Button size="xl">Extra Large</Button>
```
